rd_link

Translate a reference to a package function into a link to that function’s documentation in rd.html

This is meant to be used inside your packagedocs .Rmd files

Usage 

rd_link(txt, rd_page = "rd.html")

Arguments

txt
a string or expression referencing a function or other Rd object
rd_page
the string to be used in the href string pointing to the page where Rd documentation is provided

Details

Instead of using myfunction() when talking about a package function myfunction inside your .Rmd file, you can use rd_link(myfunction()) or even things like rd_link(myfunction(arg1 = 1, ...)) and it will turn it into an href pointing to the online documentation of your package function myfunction in the page rendered by render_docs.
